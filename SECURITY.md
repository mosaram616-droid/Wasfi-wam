# Security Policy

## Supported Versions

The following table outlines the project versions currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.3.x   | :white_check_mark: |
| < 2.3   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please follow these guidelines:

- **Do not** disclose security vulnerabilities publicly in standard GitHub issues.
- Instead, please report them by contacting the repository maintainer directly or through GitHub's private vulnerability reporting feature (if enabled in your repository's Security tab).
- Please provide a clear description of the issue, steps to reproduce, and potential impact. You will receive an acknowledgment of your report.
